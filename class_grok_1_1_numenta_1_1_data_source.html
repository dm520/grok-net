Member Data Documentation

const string Grok.Numenta.DataSource.TYPE_LOCAL = "local"

A "local" data source represents direct input into the stream. A stream must have one and only one local data source.

const String Grok.Numenta.DataSource.TYPE_PUBLIC = "public"

A "public" data source represents one of the public data providers. A stream can optionally have many data sources.
